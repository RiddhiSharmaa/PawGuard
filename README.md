🐾 PawGuard — AI-Powered Stray Dog Rescue Network
🚨 Overview

PawGuard is an AI-driven platform designed to connect people with animal rescue resources instantly. It simplifies the process of reporting injured or distressed stray dogs and ensures faster, more coordinated rescue efforts.

❗ Problem

In many cities, people struggle to help injured stray dogs because:

There’s no centralized rescue system
NGOs operate in isolated networks
Emergency response is often slow or unclear

This leads to delayed rescues and preventable suffering.

💡 Solution

PawGuard enables users to report a case in seconds by uploading a photo, description, and location.

The AI system then:

Evaluates the dog’s condition
Determines urgency
Identifies the appropriate rescue organization
Sends a request automatically

👉 No calls. No manual searching. No delays.

✨ Key Features
📸 Quick reporting with image & location
🤖 AI-based injury and urgency detection
🧠 Duplicate report prevention
📩 Automated rescue notifications
🗺️ Live tracking of reported cases
🚨 Emergency guidance for dog bites
🛠️ Tech Stack

Frontend: Next.js, TypeScript, Tailwind CSS
Backend: FastAPI, Supabase
AI System: CrewAI, Groq, Gemini Vision
Other Tools: Resend, Vercel

🧠 How It Works

PawGuard uses a multi-agent AI system where different agents handle specific tasks like analysis, validation, and rescue coordination. These agents collaborate to automate the entire workflow from report to response.
